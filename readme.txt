Easily add hidden and private notes for your users and they will never see your notes about them!

== Description ==

With this plugin, Admins can save some hidden notes about a user and the user will never see the note.



== Installation ==


1. Upload `admin-private-note` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. go to the users profile to save some notes on users...

== Frequently Asked Questions ==

= How can I save hidden notes for users?  =

just go to users profile and you will see a text editor for private noting...

= How can I see hidden notes?  =

the notes are in the users list page in Admin area...
